# Animation Guidelines

## Purpose

This document defines animation standards for the Skillar Mobile application.

## Principles

- Animations should enhance usability.
- Keep transitions smooth and purposeful.
- Respect platform conventions.
- Avoid excessive or distracting motion.

## Guidelines

- Use consistent animation durations.
- Prefer subtle transitions for navigation and state changes.
- Respect reduced motion accessibility settings where supported.
- Avoid blocking user interaction with long-running animations.

## Performance

- Keep animations lightweight.
- Minimize unnecessary re-renders during animated transitions.
- Prefer native-driven animations where appropriate.

## Future Enhancements

This guide will be expanded with motion tokens, reusable animation patterns, and examples using the project's approved animation libraries.
